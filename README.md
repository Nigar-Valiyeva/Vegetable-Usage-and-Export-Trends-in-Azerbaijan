# Exploratory Data Analysis and Time Series Analysis: Vegetable Supply Chain Dynamics in Azerbaijan (2007–2024)

## Project Overview
This project provides a comprehensive data analysis and time-series forecasting of vegetable usage, supply chain distribution, and economic export dynamics in Azerbaijan from 2007 to 2024. By applying statistical correlation and machine learning forecasting models, this analysis aims to identify historical consumption patterns, manufacturing shifts, and future agricultural economic trajectories through 2029.
